## Overview

This repository contains code for preprocessing and analysis of laminar MEG data,
with a focus on V1 evoked response fields (ERFs), as reported in *Szul et al.*.

## Requirements
- **lameg v0.1.0**

## Data availability

The raw MEG and MRI data are not publicly available.
Access to the original dataset can be requested by contacting the authors of the study.
